Readme file for the Company Management System Project:-

# Company Database

## Objective
Manage employees and their job roles, including department and salary information.

## Tables
1. Job – Contains job title, department, and salary  
2. Employee – Contains employee information and links to the job table

## Key Points
- Multi-table project: employee and job tables
- Sample data included
- 5 sample queries: joins, filtering, aggregation
- Screenshots folder included 
- Easy to run using MySQL command line
